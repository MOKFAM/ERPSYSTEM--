# 이문면옥 ERP 시스템 목표

## 개요
이문면옥의 매장 운영 전반을 통합 관리하는 내부 ERP(전사적 자원관리) 시스템 구축.
현재 Google Sheets, 문서 등에 분산된 운영 데이터를 하나의 시스템으로 통합한다.

---

## 핵심 목표

### 1. 재고 및 발주 관리
- 식재료·소모품 재고 실시간 파악
- 발주 기준량(최소재고) 설정 및 자동 알림
- 발주 이력 및 거래처별 단가 관리

### 2. 소스 OEM 관리
- 소스별 원료 배합 레시피 버전 관리
- OEM 업체 발주 및 납품 이력 추적
- 로트(lot) 단위 품질 메모 기록

### 3. 인사 및 스케줄 관리
- 직원 정보·계약 현황 관리
- 월별 근무 스케줄 편성 및 조회
- 취업규칙 기반 근태·급여 계산 연동 기반 마련

### 4. 매출 및 운영일지
- 일별 매출 기록 및 주·월 집계
- 운영 이슈·특이사항 로그
- 이전 Google Sheets 데이터 마이그레이션

### 5. 보고 및 대시보드
- 대표 보고용 주간·월간 요약 리포트
- 주요 지표(매출, 재고 소진율, 인건비 비율) 시각화

### 시스템 권한 구조

```
관리자 (Admin) — 점장, 본사 대표/이사
├── 전체 기능 접근
├── 사용자 및 시스템 설정 관리
├── 발주 최종 승인
└── 전 지점 매출/재고 조회

중간관리자 (Manager) — 실장, 운영관리자
├── 직원 근태 확인 및 휴가 승인
├── 발주 1차 검토
├── 매출/재고 조회
└── 직원 정보 조회

사용자 (User) — 주방직원, 홀직원
├── 본인 출퇴근 기록 (로그인/로그아웃)
├── 휴가/반차 신청
├── 발주 요청 등록
└── 본인 근태 내역 조회
```

### 핵심 기능 상세

#### 매출 관리
- 판매 즉시 매출 등록 → 실시간 대시보드 반영
- 일별 / 주별 / 월별 매출 통계
- 메뉴별 판매 수량 및 매출 분석

#### 재고 관리
- 메뉴 재료별 재고 등록 및 수량 추적
- 음식 판매 시 재료 재고 자동 차감
- 재고 부족 알림 기능
- 재고 실사(수동 조정) 기능

#### 인사 관리 — 직원 정보
- 입사일 기록 및 관리
- **입사 시 제출 서류 업로드:** 보건증, 주민등록등본, 통장사본, 부모님동의서(미성년자)
- 서류 업로드 시 자동 반영 (제출 여부 현황 대시보드)
- **보건증 만기 알림:** 만기일자 등록 → 갱신 시점에 자동 알림 발송

#### 인사 관리 — 출퇴근 기록
- 로그인 사용자의 출근/퇴근 시간 자동 기록 (정직원 + 아르바이트생)
- 일일 / 월간 근태 조회
- **지각, 조퇴, 결근 자동 분류** (스케줄 기준 시간 비교)
- 관리자용 전사 근태 현황 조회 (지점 전체 직원 한눈에)

#### 인사 관리 — 휴가 관리
- **연차 잔여일수 자동 계산** (입사일 기준 근로기준법 연차 부여)
- 휴가 종류: 연차, 반차(오전/오후), 병가, 경조사, 대체휴일
- **휴가 결재라인:** 사용자 신청 → 관리자 승인/반려
- 승인/반려 시 사유 기록
- **휴가 캘린더:** 전 직원 휴가 일정 캘린더 뷰로 조회
- 공휴일 자동 반영 (법정 공휴일 캘린더 연동)

#### 스케줄 관리
- **주간 스케줄:** 매주 근무 스케줄 작성 및 배포 (정직원 + 아르바이트생)
- **월간 스케줄:** 월 단위 근무 현황 조회
- 직원별 / 포지션별(주방, 홀) 스케줄 필터 조회
- 스케줄 변경 요청 및 교환 기능

#### 급여 정산
- **정직원:** 월급 기반 — 근태 기록 연동 (결근, 반차 등 반영)
- **아르바이트생:** 시급 기반 — 출퇴근 시간 기록으로 자동 근무시간 산정
- 주휴수당, 야간수당, 공휴일 수당 자동 계산
- 월별 급여 명세서 생성 및 조회
- 급여 확정 전 관리자 검토 및 승인

#### 발주 / 구매 승인
- **메뉴 재료 발주:** 사용자 요청 → 중간관리자 검토 → 대표 최종 승인
- **직원 복지 구매:** 사용 신청 → 대표 승인
- 발주 이력 조회 및 납품 확인 처리
- 승인 거절 시 사유 기록 및 재신청 가능

#### 결재 시스템 (공통)
> 발주/구매 승인, 휴가 등 승인이 필요한 모든 기능에 공통 적용

- **결재 진행 상태 추적:** 대기 → 검토중 → 승인/반려/보류 단계별 상태 표시
- **결재 이력 기록:** 누가, 언제, 어떤 결정을 했는지 전체 이력 보존
- **결재 의견 기록:** 각 결재자가 승인/반려 시 의견(코멘트) 작성 가능
- **결재 알림:**
  - 결재자에게 신규 결재 도착 알림
  - 요청자에게 결재 완료 알림
  - 요청자에게 결재 반려 알림 (반려 사유 포함)

### 인사노무 관리 (마스터 관리자 전용)

### 업무 흐름

```
[매출 흐름]
음식 판매 → 매출 등록 → 재고 자동 차감 → 대시보드 반영

[발주 흐름]
발주 요청(사용자) → 1차 검토(중간관리자) → 최종 승인(대표)
→ 구매 실행 → 재고 입고 처리

[근태 흐름]
로그인(출근) → 업무 → 로그아웃(퇴근) → 근태 기록 저장
휴가 신청(사용자) → 승인(중간관리자/관리자) → 캘린더 반영
```

### 개발 우선순위 (업데이트)

| 단계 | 모듈 |
|------|------|
| 1단계 | 사용자 인증 및 권한 시스템 (로그인, 역할별 접근 제어) |
| 2단계 | 근태 관리 (출퇴근 기록, 휴가 신청/승인) + 스케줄 관리 (주간/월간) |
| 3단계 | 급여 정산 (정직원 월급, 아르바이트 시급 자동 계산) |
| 4단계 | 매출 등록 및 실시간 대시보드 |
| 5단계 | 재고 관리 (등록, 차감, 알림) |
| 6단계 | 발주 / 구매 승인 시스템 |
| 7단계 | 보조 기능 (알림, 검색, 내보내기, 감사로그) |
| 8단계 | 본사 통합 조회 및 다지점 확장 기반 구축 |
